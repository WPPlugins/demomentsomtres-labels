== Description ==

DeMomentSomTres Labels is designed to provide product labels based on a csv file and custom html and css

= Features =
* Multiple labels can be defined.
* Icons supported.
* Each label format can requiere a capability

= History & Raison d’être =
[Xerigots](http://www.xerigots.cat/) needed a program to allow all their shops to print product labels.

This plugin is rewrited based on an ancient Python development. 

== Installation ==

This label generator can be installed as any other WordPress Plugin.

It requires DeMomentSomTres Tools plugin.

The system should be configured with allow_open directive active.
